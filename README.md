# PDF Reader Mode

## Configuration

The application can be configured using `pdfreadermode.json` file placed in the current directory or `$XDG_CONFIG_HOME` directory (usually `~/.config/pdfreadermode.json`)

Default configuration:


    {
      "document" : {
        "width" : "9.14cm",
        "height" : "12.2cm",
        "margin" : "4mm",
        "language" : "en",
        "font-size" : "10pt"
      }
    }
